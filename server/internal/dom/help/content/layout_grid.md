## Grid layout

The grid layout is a dense, sortable table. Every visible attribute is a column; clicking a header sorts; the row count fits a lot of cards on the screen at once.

**Best for:** triage, audits, or any time you need to compare many cards across the same attributes.

**Affordances:**
- Click a column header to sort ascending; click again to flip to descending.
- Inline-edit by clicking a cell; changes save on blur.
- Quick-entry at the bottom adds a new task without leaving the grid.
