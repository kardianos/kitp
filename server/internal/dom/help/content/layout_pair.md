## Pair layout

The pair layout is the project-header-plus-task-list view. It pairs a project's metadata pane (title, description, owner, key dates) with a flat list of every task in that project.

**Best for:** a project landing page — somewhere a stakeholder can read the project context and immediately see what is in flight.

**Affordances:**
- The header is editable for users with project-edit rights; other users see a read-only view.
- The task list under the header uses the same row affordances as the list layout (reorder, transition bar, quick-entry).
